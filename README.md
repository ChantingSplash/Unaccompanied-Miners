This is what could be considered my capstone for Computer Game Design and Development.
This a short game where you play as a miner who needs to get himself out of debt.
Venture into the dangerous mines to collect gems to pay back all you owe the eponimous Company.
The mines contain both riches and dangers. Goblins patrol every turn and beware the Wolves.

I built the entire UI system for the game including all level UI and the title screen. 
I also built the system we used to save our data between play sessions. 
I did not build the level designs. However, I implemented the method that the game uses to load the layouts into each level.
